# Desperado

集中することを……強いられているんだ！

for OS X Lion

## 使い方

    $ gem install desperado
    $ sudo desperado -b twitter.com

これでTwitterにはアクセスできなくなりました。よかったですね。

    $ sudo desperado -u twitter.com

解除は `-u` もしくは `--unblock` 。追加は `-b` もしくは `--block` です。

## 指定しておくと便利なサイト

- twitter.com
- facebook.com
- mixi.jp
- pixiv.net
- nicovideo.jp
  - これだけだとブロックしきれないので www.nicovideo.jp も
- youtube.com
- www.youtube.com
